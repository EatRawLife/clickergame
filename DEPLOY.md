# 배포 가이드

## 방법 1: Netlify Drop (가장 간단)

1. https://app.netlify.com/drop 접속
2. 이 폴더의 파일들(index.html, style.css, script.js)을 드래그 앤 드롭
3. 자동으로 공개 URL이 생성됩니다!

## 방법 2: GitHub Pages

1. GitHub 저장소 생성
2. 파일들을 업로드
3. Settings > Pages에서 배포 활성화
4. `https://[사용자명].github.io/[저장소명]` 으로 접속

## 방법 3: Vercel

1. https://vercel.com 접속
2. 프로젝트 import
3. 자동 배포 완료

## 현재 상태 (로컬 서버)

- `http://localhost:8000` - 본인 컴퓨터에서만 접근
- `http://192.168.0.31:8000` - 같은 Wi-Fi 네트워크에서만 접근
